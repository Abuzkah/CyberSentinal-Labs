# Phishing Awareness Simulator

A non-malicious Python tool to generate mock phishing emails for user awareness training.

## Objective
Help users recognize phishing attempts by analyzing simulated emails.

## How it Works
- Generates realistic phishing email samples
- Analyzes emails for common phishing indicators
- No emails are sent; for local demo only

## How to Run
```sh
python phishing_simulator/main.py
```

## Requirements
- Python 3.x
